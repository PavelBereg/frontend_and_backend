# Практическая работа №10: Валидация форм и события

В рамках работы была реализована клиентская валидация формы обратной связи на странице «Контакты» проекта **CodeStart**.

### Основные функции:
*   **Валидация данных:** Проверка ФИО (минимум 2 слова), телефона (не менее 10 цифр), корректности Email и обязательного согласия с правилами.
*   **Визуальный отклик:** Использование классов фреймворка Bulma (`is-danger`) и динамическое добавление текста ошибок под полями.
*   **Работа с событиями:** Разделение логики на два независимых скрипта: `validation.js` проверяет данные и создает событие `formValid`, а `consoleLogger.js` отлавливает его и выводит информацию в консоль браузера.

**Технологии:** JavaScript (CustomEvents, DOM API), Bulma CSS.
